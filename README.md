# TransitOps — Smart Transport Operations Platform

A centralized fleet management platform for tracking vehicles, drivers, trips, maintenance, and fuel expenses with automated business rule enforcement.

## 🚀 Live Demo
👉 https://transit-ops-odoo-hackathon.vercel.app

## ✨ Features
- Firebase Authentication (Email/Password + Google)
- Real-time Dashboard with fleet KPIs
- Vehicle Registry with unique registration validation
- Driver Management with license expiry tracking
- Trip Dispatcher (Draft → Dispatched → Completed/Cancelled lifecycle)
- Maintenance workflow with automatic vehicle status sync
- Fuel & Expense tracking
- Analytics with fleet utilization and cost metrics
- Fully responsive (mobile & desktop)

## 🛠️ Tech Stack
- React + Vite
- Firebase (Auth + Firestore)
- Tailwind CSS
- Deployed on Vercel
